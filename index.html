What is Greenscape?

Greenscape is basically an web portal framework based on modern architecture like REST. The server exposes a REST API for client to fetch data. The client is built on moder web technologies like HTML5, AngularJS and Foundation css library. The server is completely modular using the the robust OSGi framework.
